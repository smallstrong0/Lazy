### Lazy是一个Python脚本处理UI设计师给我们用sketch切出的图进行一键更名，批量压缩的工具。----- 解放双手，值得拥有。

> Use

1. 将工程中的3个Python文件copy到UI设计师给的hdpi,xhdpi,xxhdpi，xxxhdpi的同级目录下。

2. 根据提示依次运行，python list.py  python cname.py  python tinypng.py。  

3. 欢迎star
